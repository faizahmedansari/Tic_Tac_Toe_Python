# 🎮 Tic Tac Toe (Python)

A colorful Tic Tac Toe game built using Python and Tkinter.

## Features

- 🎨 Colorful GUI
- 👥 Two-player mode
- 🏆 Winner detection
- 🤝 Draw detection
- 🔄 Restart game button

## Technologies Used

- Python
- Tkinter

## How to Run

bash
python tic_tac_toe.py
